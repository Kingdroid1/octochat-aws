<h1 align="center">:smile_cat:<tt>octochat</tt></h1>

## Usage

Build the container:

```shell
docker build -t octochat .
```

Run the container:

```shell
./script/server.sh
```

## Development usage

Build the container:

```shell
docker build -t octochat -f Dockerfile.develop .
```

Run the container:

```shell
./script/server.sh
```
